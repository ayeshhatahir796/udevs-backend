# udevs-backend

This is a Node.js backend server built using Express and PostgreSQL.

## 🚀 Features
- User Registration & Login APIs
- PostgreSQL Database Integration
- MVC Folder Structure

## 📁 Project Structure

udevs-backend/

├── .env

│ └── db.js
├── controllers/

├── models/
│ └── userModel.js
├── routes/
│ └── userRoutes.js
